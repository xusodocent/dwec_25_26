    Exemple d'ús del mètode map()

    El mètode array.map()

    El mètode map en JavaScript és un mètode d'arrays que es fa servir per a crear un nou array transformant els elements d'un array original segons una funció que li passes.

    Aquest mètode vos pot servir per a resoltre un dels exercicis de les pràctiques...

    Enunciat

    Genera una matriu on cada fila serà el nom d'un producte i el seu preu en euros. Posa el preu entre 0 i 100€. Aplica les funcions necessàries per a, a partir d'aquesta matriu, obtindre altra matriu amb aquells productes que el seu preu siga major de 75€ i s'haja aplicat una rebaixa del 10%.

    // Funció per a generar un preu aleatori entre 0 i 100€
function generarPreu() {
    return Math.floor(Math.random() * 101); // Preus entre 0 i 100
}

// Llista de productes
const productes = ['Producte A', 'Producte B', 'Producte C', 'Producte D', 'Producte E'];

// Generar la matriu de productes amb preus aleatoris
let matriuProductes = productes.map((producte) => {
    return [producte, generarPreu()];
});

console.log("Matriu original:", matriuProductes);

// Filtrar productes amb preu > 75€ i aplicar una rebaixa del 10%
let matriuProductesRebaixats = matriuProductes
    .filter(([producte, preu]) => preu > 75)
    .map(([producte, preu]) => [producte, preu * 0.9]);

console.log("Productes amb preus > 75€ amb rebaixa del 10%:", matriuProductesRebaixats);

    Funcionament del map:

    • Entrada: El mètode map pren una funció com a argument, la qual s'aplica a cada element de l'array original.

    • Transformació: Aquesta funció processa o modifica cada element de l'array i retorna un nou valor.

    • Resultat: Com a resultat, obtens un nou array amb els elements transformats, però l'array original es manté intacte (no es modifica).

    • index (opcional): L'índex de l'element actual en la iteració.

    • array (opcional): El mateix array original que s'està iterant.

    Exemple simple:

    Imagina't que tens un array de números i vols multiplicar cada número per 2.

    let numeros = [1, 2, 3, 4];
let multiplicats = numeros.map((numero) => numero * 2);

console.log(multiplicats); // [2, 4, 6, 8]

    Ací:

    • L'array numeros és [1, 2, 3, 4].

    • El map aplica la funció (numero) => numero * 2 a cada element.

    • El resultat és un nou array: [2, 4, 6, 8].

    El map s'utilitza per:

    • Iterar sobre la matriu filtrada (que conté els productes amb preu > 75€).

    • Transformar cada producte aplicant una rebaixa del 10% al seu preu: [producte, preu * 0.9].

    • Retornar un nou array on cada producte té el seu preu rebaixat.

    En resum, el mètode map serveix per a transformar cada element d'un array segons les teves necessitats, creant-ne un de nou.
