    DWEC VUE router practica semiguiada

    Gestió de cursos de programació

    Aquesta pràctica guiada consisteix en la creació d'una aplicació anomenada "Gestor de Cursos de Programació Web", pensada per a gestionar cursos de programació. Aprendreu a utilitzar Vue 3 per a desenvolupar un CRUD (Crear, Llegir, Actualitzar i Esborrar) integrant json-server com a API backend. També aprendreu a crear components reutilitzables, fer ús del sistema de rutes de Vue Router i gestionar l'estat local dins de l'aplicació.

    Requeriments del projecte

    1. Frontend: Vue 3.
    2. Backend: json-server per a simular una API REST.
    3. Característiques:
    4. Llistar cursos: Mostrar tots els cursos disponibles.
    5. Afegir cursos: Formulari per a crear un nou curs.
    6. Editar cursos: Modificar la informació d'un curs existent.
    7. Esborrar cursos: Eliminar cursos.
    8. Utilitzar Vue Router per a la navegació entre pàgines:
    9. / - Llistat de cursos.
    10. /nou - Formulari per a afegir un curs.
    11. /editar/:id - Formulari per a editar un curs.

    Estructura del projecte

    Components

    1. Components bàsics:
    2. CourseItem.vue: Representa un curs individual.
    3. CourseList.vue: Llista tots els cursos utilitzant CourseItem.vue.

    4. CourseForm.vue: Formulari reutilitzable per a crear i editar cursos.

    5. Component principal:

    6. CourseManager.vue: Gestiona la vista principal i inclou CourseList.vue.

import Home from '../views/Home.vue';
import AddCourse from '../views/AddCourse.vue';
import EditCourse from '../views/EditCourse.vue';

const routes = [
  { path: '/', component: Home },
  { path: '/nou', component: AddCourse },
  { path: '/editar/:id', component: EditCourse },
];

const router = createRouter({
  history: createWebHistory(),
  routes,
});

export default router;

      <template>

<script>
import CourseItem from './CourseItem.vue';

export default {
  components: { CourseItem },
  props: {
    courses: Array
  },
  methods: {
    editCourse(id) {
      this.$router.push(`/editar/${id}`);
    },
    deleteCourse(id) {
      this.$emit('delete', id);
    }
  }
};
</script>

  props: {

  emits: ['submit'],
  methods: {
    submit() {
      this.$emit('submit', this.course);
    }
  }
};
</script>
