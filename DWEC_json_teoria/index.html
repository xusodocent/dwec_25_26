    Tutorial JSON

    Què és JSON i tipus de dades

    JSON

    JSON (JavaScript Object Notation) és un format lleuger d'intercanvi de dades que es representa com a text estructurat. Es basa en parelles clau: valor i és utilitzat per enviar i rebre dades entre clients i servidors. És fàcilment llegible tant per humans com per màquines.

    Tipus de dades que pot tenir un JSON

    1. Objecte: Un conjunt de parelles clau-valor envoltat per {}.
    2. Array: Una llista ordenada de valors separats per comes, envoltada per [].
    3. Cadena de text (string): Valors envoltats amb cometes dobles "".
    4. Nombre: Pot ser un nombre enter o decimal.
    5. Booleà: true o false.
    6. Null: Representa l'absència de valor.

    const persona = {
  nom: "Joan",             // String
  edat: 30,                // Nombre
  habilitats: ["HTML", "CSS", "JavaScript"], // Array
  actiu: true,             // Booleà
  direccio: {              // Objecte

  }
};

// Cridar el mètode
console.log(persona.mostrarHabilitats());
// Resultat: Les habilitats de Joan són: HTML, CSS, JavaScript.

    Explicació:

    • El mètode mostrarHabilitats està definit dins de l'objecte persona utilitzant la paraula clau function.

    • Per accedir a les propietats dins del mateix objecte, fem servir this per referir-nos a l'objecte actual (persona).

    • Quan crides persona.mostrarHabilitats(), el mètode retorna un missatge amb les habilitats de la persona.

    Amb aquesta tècnica, pots definir mètodes que treballen directament amb les propietats de l'objecte on estan definits.

    Diferència entre el llenguatge JSON i els JSON en JavaScript

    1. JSON (JavaScript Object Notation)

    JSON és un format de dades que es basa en text i està dissenyat per intercanviar dades de manera estructurada entre sistemes. Encara que el seu nom fa referència a JavaScript, JSON no depèn del llenguatge i pot ser utilitzat per molts altres llenguatges de programació com Python, Java, PHP, etc.

    • Característiques clau de JSON com a llenguatge de dades:

    • És independent del llenguatge de programació.

    • El seu format és molt semblant a la sintaxi dels objectes literals de JavaScript, però és només text.
    • Només admet determinats tipus de dades: objectes, arrays, strings, nombres, booleans, i null.
    • No admet mètodes o funcions dins d’un JSON.
    • S’utilitza principalment per intercanviar dades entre un servidor i un client o entre sistemes.

    Exemple de JSON (només dades en format text):

    JSON
    1

    {
  "nom": "Joan",
  "edat": 30,
  "habilitats": ["HTML", "CSS", "JavaScript"],
  "actiu": true
}

    Només text

    Aquest JSON és només text i no pot executar cap funció o lògica.

    2. JSON en JavaScript

    En JavaScript, un objecte JSON es refereix a un objecte literal de JavaScript que segueix la mateixa estructura que un JSON.

    Inclou funcions

    A diferència del format JSON (que és text), els objectes JavaScript poden incloure funcions i altres comportaments.

    Diferències Clau:

    Característica JSON (Format de Dades) Objecte JSON en JavaScript
    Tipus Format de text Objecte literal dins de JavaScript
    Funcions No pot contenir funcions Pot contenir funcions o mètodes
    Utilitat Intercanvi de dades entre sistemes Manipulació directa d'objectes en JS
    Tipus de dades Limitat a tipus bàsics (objecte, array, string, número, booleà, null) Tots els tipus de JavaScript (inclou funcions)
    Conversió Necessita JSON.parse() per ser utilitzat en JS Ja és un objecte JavaScript

    En resum, JSON com a format de dades és només text que es pot utilitzar per intercanviar dades entre sistemes, mentre que els objectes JSON en JavaScript són objectes reals que poden incloure lògica i poden ser manipulats directament pel codi.

    Exemples i utilitats de JSON.parse() i JSON.stringify()

    1. JSON.parse()

    JSON.parse() és una funció en JavaScript que s’utilitza per convertir una cadena JSON (text) en un objecte JavaScript. Això és especialment útil quan reps dades d’un servidor en format JSON i vols treballar amb elles com a objectes en JavaScript.

    Exemple 1: Conversió de cadena JSON a objecte

    Imagina que reps una resposta d'un servidor que retorna dades en format JSON:

    JavaScript
    1
2
3
4
5
6
7
    const jsonString = '{"nom": "Joan", "edat": 30, "ciutat": "Barcelona"}';

// Convertim la cadena JSON a un objecte JavaScript
const persona = JSON.parse(jsonString);

console.log(persona.nom);  // Resultat: Joan
console.log(persona.edat); // Resultat: 30
    • Utilitat: Permet convertir una cadena de text que prové d'una API o un fitxer JSON en un objecte que pots utilitzar en JavaScript.
    Exemple 2: Dades complexes des de l'API

    2. JSON.stringify()

    JSON.stringify() és una funció en JavaScript que s’utilitza per convertir un objecte JavaScript a una cadena JSON. És molt útil quan vols enviar dades a un servidor o emmagatzemar objectes locals com a text, per exemple en localStorage.

    Exemple 1: Conversió d'objecte JavaScript a JSON

    Imagina que tens un objecte JavaScript i vols convertir-lo a una cadena JSON:

  edat: 30,
  ciutats: ["Barcelona", "Madrid"]

const jsonString = JSON.stringify(persona);

console.log(jsonString);
// Resultat: {"nom":"Joan","edat":30,"ciutats":["Barcelona","Madrid"]}
    • Utilitat: Això és útil quan vols enviar dades en format JSON a una API o guardar objectes com a text en bases de dades o localStorage.

function mostrarPorts() {
  const contenidor = document.getElementById("ports");

  // Recorrem cada serralada (Pirineus, Alps, Pics d'Europa)
  for (const serralada in portsMuntanya) {
    const ports = portsMuntanya[serralada];

    // Afegim el títol de la serralada
    const titolSerralada = document.createElement("h2");
    titolSerralada.textContent = serralada.charAt(0).toUpperCase() + serralada.slice(1);
    contenidor.appendChild(titolSerralada);

    // Recorrem els ports dins de cada serralada
    ports.forEach(port => {
      const divPort = document.createElement("div");
      divPort.classList.add("port");

      // Afegim la imatge del port
      const imatge = document.createElement("img");
      imatge.src = port.url_foto;
      imatge.alt = port.nom;
      divPort.appendChild(imatge);

      // Afegim el nom del port
      const titol = document.createElement("h3");
      titol.textContent = port.nom;
      divPort.appendChild(titol);

      // Afegim altres detalls del port
      const detalls = document.createElement("p");
      detalls.innerHTML = `
        <span class="title">Altitud:</span> ${port.altitud} m<br>
        <span class="title">Desnivell:</span> ${port.desnivell} m<br>
        <span class="title">Dificultat:</span> ${port.dificultat}<br>
        <span class="title">Localització:</span> ${port.localitzacio}
      `;
      divPort.appendChild(detalls);

      // Afegim el div del port al contenidor principal
      contenidor.appendChild(divPort);
    });
  }
}

// Executem la funció per mostrar els ports a la pàgina
mostrarPorts();
